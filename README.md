# tampermonkey_script

## 博客网站的精简和宽屏处理(blogs)

- 处理一些干扰阅读文章的元素, 提供更好的阅读体验
- 屏蔽复制需要登录的要求, 方便复制内容
- 优化外链的访问, 避免打开外链还要多跳转一次的情况
- 针对宽屏进行宽度调整, 让每行可以显示更多内容
- 关于宽屏调整: 如果觉得宽屏调整后左右跨度太大不方便阅读的话, 可以在脚本的设置里关闭宽屏调整
